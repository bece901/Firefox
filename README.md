#####about:config :
* 搜索 view_source.editor.path 添加自己的编辑器路径，形如："C:\Program Files (x86)\Notepad++\notepad++.exe"
* 搜索 extensions.greasemonkey.installDelay 将值改为0

#####Adblock List :
* [BeceADPList](http://notepad.cc/beceadplist)：自用过滤列表备份
